# Covid_Tracker_App

A completely responsive Covid_Tracker App- Works on Android & IOS

- WorldStatesRecord-Api :- https://disease.sh/v3/covid-19/all
- CountriesList-Api :- https://disease.sh/v3/covid-19/countries
